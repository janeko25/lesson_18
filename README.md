# Vioty Ice Cream

Сайт-вітрина для бренду морозива **Vioty Ice Cream**. Проект показує продукцію, процес замовлення та контакти.

## Структура проекту

/index.html         — головна сторінка  
/style.css          — стилі сайту  
/img/               — зображення з макету та логотипи  
/img.my/            — твої згенеровані зображення морозива та людей

## Секції сайту

1. **Header**  
   - Логотип  
   - Навігаційне меню  
   - Соцмережі (Instagram, Facebook, TikTok, YouTube)

2. **Головна секція**  
   - Основний заголовок (`<h1>`)  
   - Кнопка “Buy Now”  
   - Зображення головного морозива  

3. **Про наше морозиво**  
   - Інформація про походження продукту  
   - Фото людей, що їдять морозиво  

4. **Наші продукти**  
   - Картки продуктів: Strawberry, Chocolate, Vanilla, Mango  
   - Кожна картка має: `<h3>`, опис, ціну та кнопку “Buy Now”

5. **Як замовити**  
   - Покрокова інструкція замовлення  
   - Кнопки для завантаження додатку  

6. **Запитання**  
   - Форма для запитань користувачів  

7. **Footer**  
   - Про компанію  
   - Контакти, локація  
   - Соцмережі

## Технології

- HTML5  
- CSS3  
- Адаптивна верстка (viewport meta)

## Коміти

1. Додана секція “Наші продукти”  
2. Додана секція “Як замовити”  
3. Додана секція “Запитання” (форма для користувачів)  
4. Доданий футер з контактами та соцмережами

## Зображення

- `/img/` — макетні зображення  
- `/img.my/` — твої згенеровані зображення

## Примітки

- Використовуються семантичні теги `<header>`, `<main>`, `<section>`, `<article>`, `<footer>`  
- Один `<h1>` на сторінку, інші заголовки `<h2>`–`<h3>`  
- Всі `<img>` теги без trailing slash для валідності HTML5  
- Кнопки поза формою мають `type="button"`, у формі `type="submit`
